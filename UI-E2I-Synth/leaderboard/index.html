UI-I2E-Bench Leaderboard

UI-I2E-Bench is a benchmark for GUI visual grounding. It incorporates implicit instructions and long-tail UI element types, with element-to-screen size ratios that better reflect real-world scenarios.
This page presents both a detailed view of UI-I2E-Bench results and a comparison across multiple benchmarks (Combined).
